CLUB MEMBER DATABASE

FILTERS

User Guide:

  • Picking "..." from the PLAYER SELECT drop down menu will return you to this guide.
  • Picking "All" from the PLAYER SELECT drop down will show all available club member garages.
  • Picking any specific player from the PLAYER SELECT drop down will show only that player's garage.
  • ★★★ stars indicate empty stars/total stars.
  • ✮✮✮ stars indicate the car's current star level.
  • ✮✮✮ stars indicate car is golden/maxed.
  • Search bar can be used to search for car(s) in either one player's garage or all player garages depending on PLAYER SELECT selection.
  • Use BRAND NAME drop down menu to filter required brand of car(s). Result will show either from one garage or from all garages depending on PLAYER SELECT selection.
  • Using the STARS dropdown menu will filter cars by their current star level.
  • Using the CLASS dropdown menu will filter cars by their class.
  • Checking the CAR MAXED checkbox will filter only maxed/golden cars from the garage(s)
  • Checking the UNLOCKED CARS ONLY checkbox will filter only cars unlocked by the player(s)
